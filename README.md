# Student-Performance-Attendance-Analytics-Dashboard-
Student Performance &amp; Attendance Analytics Dashboard 
 
Student Performance & Attendance Analytics — Power BI Case Study
This repository contains the full case study, documentation, and files for the Student Performance Dashboard project built using Power BI.

Case Study Overview
The project analyzes:
Internal-1 & Internal-2 marks
Final examination scores
Subject-wise performance
Monthly attendance trends
Final percentage & grade
The purpose is to give faculty a complete interactive view of a student’s academic progress.
